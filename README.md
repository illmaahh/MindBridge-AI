# MindBridge AI 🧠💙

## Overview

MindBridge AI is a multi-agent mental wellness assistant that analyzes user mood, provides insights, and suggests personalized recommendations.

## Features

* AI-powered chat support
* Mood analysis
* Journal analysis
* Emotional insights tracking
* Personalized recommendations

## Tech Stack

* Python
* FastAPI
* Jinja2 Templates
* AI Agents Architecture

## Live Demo 🌍

https://mindbridge-ai.onrender.com

## How to Run Locally

```bash
cd mindbridge
pip install -r requirements.txt
python main.py
```
